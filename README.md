# PokeDex 

Goal - To build a functional PokeDex using Poke-API 




The PokeDex ususes an API to fetch Pokemons. 
Once a Pokemon is selected from the "modal"(list), a pop modal shows with the Pokemons :
- Name
- Height
- Type
- Ability 
- Image of Pokemon
